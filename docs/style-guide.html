08

상호작용 상태 & 포커스

맥·윈도(마우스+키보드)와 아이폰(터치)을 함께 쓰므로 상태를 명시. 키보드 포커스 링은 필수 — 골드 반투명 링 --focus-ring.

default
hover · bg 살짝 진하게
active · 1px down
focus-visible · 링
disabled · opacity/회색
· hover: 배경 한 단계 진하게(테두리 버튼) / CTA는 밝기 -6%. 터치 기기에선 무시됨 — hover에 필수 정보 두지 말 것
· active: transform:translateY(1px) 로 눌림감
· focus-visible: box-shadow:var(--focus-ring) — :focus 아닌 :focus-visible로 걸어 마우스 클릭엔 안 뜨게
· disabled: 회색 배경 + ink-faint 텍스트 + cursor:not-allowed. 색만으로 구분 금지 → 클릭 불가 시각 신호 병행
.btn:hover{background:#F3F2EC;border-color:#C9CBC3}
.btn:active{transform:translateY(1px)}
.btn:focus-visible{outline:none;box-shadow:var(--focus-ring)}
.btn:disabled{background:#F7F6F1;color:var(--ink-faint);
    border-color:var(--hairline-soft);cursor:not-allowed}

13

Do / Don't — 골드 텍스트

✕DON'T

골드 #C7A14A 본문 텍스트

흰 배경 대비 2.43:1 — 작은 글씨·링크에서 안 읽힘.

✓DO

line9-deep #9A7B2E — 큰/굵은 라벨

골드는 아이콘·배지 배경에, 텍스트는 line9-deep. 작은 본문은 ink/ink-soft.

14

Claude Code 반영 순서

  1. 토큰 먼저: :root에 여백(lg/xl/2xl)·타이포·그림자·포커스링 토큰 추가. 색·기존 반경 값은 그대로.
  2. 반경 정리: border-radius:9px~12px 잔여 → 역할 토큰으로 치환(버튼 md / 칩 sm / 카드 card / pill).
  3. 골드 텍스트 감사: color:#C7A14A 텍스트 용례 → line9-deep 또는 ink로. 골드는 아이콘·배경만.
  4. 아이콘 교체: .ic 클래스 + 7종 SVG 삽입, 매핑표대로 이모지 치환. ✅❌➕🔄 → 텍스트 기호.
  5. 매물 카드: 장식 아이콘 제거, 정보 행에 무채색 행 표식만. 상태칩 dot+테두리로.
  6. 상태/포커스: :hover / :active / :focus-visible / :disabled 규칙을 버튼·입력에 적용.
  7. 모바일 툴바: 매물탭 6개 → 상시 2개(정렬·추가) + ⋯ 더보기 드롭다운(기존 컴포넌트 재사용).
